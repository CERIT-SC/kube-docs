# VPN

Secure VPN je založen na [WireGuard](https://www.wireguard.com/).

## Připojení k Secure VPN

### Windows

   1. Nainstalujte WireGuard, stáhněte instalační program pro Windows [zde](https://www.wireguard.com/install/)
   2. Vytvořte konfigurační soubor `sensitive-vpn.conf` ve svém osobním, bezpečném adresáři, např. `C:\Users\<uživatelské_jméno>\Documents\sensitive-vpn.conf` s obsahem, který jste získali od našeho týmu. Příklad konfiguračního souboru:

   ```
    [Interface]
    PrivateKey = DESKTOP_CLIENT_PRIVATE_KEY
    Address = 10.0.0.2/24

    [Peer]
    PublicKey = SERVER_PUBLIC_KEY
    Endpoint = SERVER_IP_ADDRESS:PORT
    AllowedIPs = 0.0.0.0/0
   ```
   3. Spusťte aplikaci `WireGuard`.
   4. V aplikaci `WireGuard` vyberte `Import tunnel(s) from file` a vyberte `sensitive-vpn.conf`, který jste uložili ve 2. kroku.   
   ![wg1](/img/ceritsc/wg1.jpg)   
   5. Pro aktivaci tunelu `(1) Vyberte tunel, který chcete aktivovat` a `(2) Aktivujte tunel`
   ![wg2](/img/ceritsc/wg2.jpg)
   6. Tunel je aktivován, pro deaktivaci tunelu použijte tlačítko `Deactivate`.
   ![wg3](/img/ceritsc/wg3.jpg)

### Linuxové operační systémy

   1. Nainstalujte WireGuard, stáhněte pro Linux [zde](https://www.wireguard.com/install/)
   2. Vytvořte konfigurační soubor `/etc/wireguard/wg0.conf` s obsahem, který jste získali od našeho týmu.   
   Příklad:

   ```
    [Interface]
    PrivateKey = DESKTOP_CLIENT_PRIVATE_KEY
    Address = 10.0.0.2/24

    [Peer]
    PublicKey = SERVER_PUBLIC_KEY
    Endpoint = SERVER_IP_ADDRESS:PORT
    AllowedIPs = 0.0.0.0/0
   ```
   3. Pro aktivaci tunelu zadejte do `terminálu` a použijte následující příkaz:   
   ```
   wg-quick up wg0
   ```
   4. Pro kontrolu statistik tunelu (pokud je tunel aktivní) použijte:
   ```
   wg-quick show
   ```
   5. Pro deaktivaci tunelu použijte:
   ```
   wg-quick down wg0
   ```
